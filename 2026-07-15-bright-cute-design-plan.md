# 십 묶음 택배 터미널 밝고 귀여운 디자인 개선 계획

**작성일:** 2026-07-15  
**목표:** 검증된 9개 미션과 접근성 흐름은 유지하면서, 초등 저학년 학생이 편안하고 즐겁게 사용할 수 있는 밝은 종이 공예형 택배 놀이터로 화면 전체를 통일합니다.

## 1. 디자인 원칙

- 크림색 바탕 위 하늘색·민트·살구색·노란색을 사용해 밝은 인상을 강화합니다.
- 카드와 버튼은 둥근 모서리, 굵은 외곽선, 짧고 부드러운 그림자를 사용합니다.
- 귀여운 장식은 학습 정보와 겹치지 않게 배경·배지·아이콘에 한정합니다.
- 색상만으로 상태를 구분하지 않고 글자, 테두리, 체크 표시를 함께 유지합니다.
- 버튼은 최소 48px, 본문 대비와 키보드 초점 표시는 현재 수준 이상으로 유지합니다.

## 2. 화면 개선

### 공통 화면

- 헤더를 둥근 흰색 플로팅 카드로 바꾸고, `10` 상자 로고에 귀여운 표정을 더합니다.
- 배경에 구름·별·포장 스티커를 연상시키는 저대비 CSS 장식을 추가합니다.
- 제목, 안내 카드, 버튼에 일관된 스티커형 외곽선과 색상 체계를 적용합니다.
- 주요 버튼은 밝은 파란색, 현재 행동은 살구색·노란색 강조로 구분합니다.

### 시작 화면

- 기존 생성 삽화는 유지하고, 카드 주변을 파스텔 물결과 스티커 배지로 정리합니다.
- `연습 → 배송 → 정리`의 짧은 학습 여정을 한눈에 보여 줍니다.
- CTA와 안내 칩을 더 크고 명확한 저학년용 형태로 개선합니다.

### 연습·미션 화면

- 십 묶음과 낱개 영역을 파스텔 선반 카드처럼 표현합니다.
- 묶음 상자와 낱개 상자에 더 둥근 형태와 따뜻한 테이프 질감을 적용합니다.
- 현재 할 일, 성공, 경고 상태가 서로 다른 배경·아이콘·테두리로 즉시 구분되게 합니다.

### 완료·정리 화면

- 완료 화면에 종이 꽃가루 느낌의 CSS 장식을 추가합니다.
- 숫자·읽기·활동 카드의 대비와 간격을 통일하고 선택 상태를 더 분명하게 표현합니다.

## 3. 구현 범위

- `app/globals.css`: 전체 팔레트, 카드, 버튼, 택배 모형, 반응형, 모션 개선
- `app/components/AppHeader.tsx`: 귀여운 상자 로고와 헤더 구조 보강
- `app/components/LearningScreens.tsx`: 시작 화면 학습 여정과 배지 추가
- `app/components/ParcelVisual.tsx`: 영역 아이콘과 시각 라벨 개선
- `app/components/TenBundleTerminal.tsx`: v1.2.0 업데이트 내역 추가
- `public/og.png`, `app/layout.tsx`: 새 디자인과 맞는 공유 카드 반영

## 4. 검증 및 배포

- 린트, 타입 검사, 단위·컴포넌트 테스트, 24개 브라우저 흐름 테스트, 빌드를 통과합니다.
- 320px·375px·768px·1280px에서 가로 넘침이 없는지 기존 검사를 유지합니다.
- 모든 코드 파일을 500줄 미만으로 유지합니다.
- 커밋·푸시 후 기존 공개 Sites 프로젝트에 새 버전으로 배포합니다.
